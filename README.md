# react
Proyecto para diplomado Full Stack IPSS modulo 2

# Nombre del proyecto
Tienda Todo Balato con productos por categoria

# Breve descripción del e-commerce
Página web de productos en linea para compra, con consumo de API desde https://dummyjson.com/docs/products

# Componentes creados
Navbar
Categoria
Button
Footer
Product
ProductList


# Instrucciones para ejecutar el proyecto
- clonar repositorio https://github.com/ingoose-dev/ecommerce.git 
- npm install
- npm run dev

# Tecnologías usadas
vite
react
